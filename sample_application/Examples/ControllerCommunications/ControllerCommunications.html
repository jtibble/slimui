Controller Communication

The SlimUI Framework provides a service that lets controllers communicate on a shared channel. Include the ControllerCommunication service as a parameter in your controller to begin using it: 
var FunctionName = function(Context, ControllerCommunication){
The ControllerCommunication service has four methods available: 
ControllerCommunication.registerCallback(channel, callback);
ControllerCommunication.notifyObservers(channel);
ControllerCommunication.get(channel);
ControllerCommunication.set(channel, data);

get(channel) retrieves the data associated with a particular 'channel'.

set(channel, data) sets the data on a particular channel and then calls notifyObservers() to alert all listeners to the changed data.

registerCallback(channel, callback) associates a function in your controller with a ControllerCommunication 'channel'. When the data associated with that channel changes, this callback function will be invoked.

notifyObservers(channel) notifies all listeners on a given 'channel' that the data has changed. Calling ControllerCommunication.set() calls this automatically.
