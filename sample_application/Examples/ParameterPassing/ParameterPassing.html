Parameter Passing

Notice the URL for this page! This page is not part of Examples, but its own top-level route.

URL Parameter-passing can be used to select a particular item and drill down into its details.

To pass paramters to a view, you must add "parameters": true to the view in your config.json file: 

"ParameterPassing": {
	"url": "ParameterPassing",
	"template": "ParameterPassingExampleTemplate",
	"model": "ParameterPassingModel",
	"modelbuilder": "ParameterPassingModelbuilder",
	"actions": "ParameterPassingActions",
	"controller": "ParameterPassingController",
	"parameters": [
		"parameterExampleId",
		"parameterExampleId2"
	]
}

To demonstrate URL parameter-passing, add text to the end of the URL, or click this link:
#/ParameterPassing/parameter1/parameter2

To demonstrate URL parameter-passing in both a parent and subroute, click this link:
#/ParameterPassing/parameter1/parameter2/ParameterPassingSubroute/parameter3/parameter4
