Example:

Begin by creating a new view in your config.json file: 

{
    "applicationConfig": {
        "views": {
            "Simple": {
                "url": "Simple",
                "template": "SimpleTemplate",
                "model": "SimpleModel",
                "modelbuilder": "SimpleModelbuilder",
                "actions": "SimpleActions",
                "controller": "SimpleController"
            }
        }
    }
}

Next, add the files to your properties section: 

{
    "applicationConfig": {
        "properties": {
            "jsPaths": {
                "SimpleModel": "path/to/SimpleModel.js",
                "SimpleModelbuilder": "path/to/SimpleModelbuilder.js",
                "SimpleActions": "path/to/SimpleActions.js",
                "SimpleController": "path/to/SimpleController.js"
            },
            "templatePaths": {
                "SimpleTemplate": "path/to/Simple.html"
            }
        }
    }
}
