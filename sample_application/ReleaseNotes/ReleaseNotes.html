Release Notes

v0.0.7 Stable March 21, 2014

  • Rebuilt Parameter Passing functionality with examples
  • Created this Release Notes page
  • Config.json: Restructured and simplified. Added assumptions about file names and naming conventions to automate loading files, which greatly reduces config.json complexity. Added support for multiple root-folders, specified on a view-by-view basis. Updated config.js examples and documentation.

v0.0.6 March 20, 2014

  • Refactored framework to separate Model, Modelbuilder, Actions, and Controller. See Simple Page Documentation.
  • Reworked route-generation to allow for view to have a url of '/'
  • In config.json, renamed 'regions' to 'subroutes'

v0.0.5 March 14, 2014

  • Changed 'FrameworkPOST' service to 'FrameworkAJAX'
  • Removed 'Session' service
  • Removed two 'Routing' service methods
