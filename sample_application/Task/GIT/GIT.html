1.-	First, we need to go to local disk, we can use the next syntaxis :  
		cd ../..   (click enter)
2.-	Now, go to workspace:
		cd GE_DATA_CLASSIFICATION/confidencial/workspace/  (click enter)
3.-	Type your OpenGE mail	
		git clone ssh://nameaccount@ge.com/gitroot/ls2_0 ls20angular  (click enter)

4.-	Then, in the screen will be appear a message dialog with two options:

5.-	When the download finish, type:
		cd ls20angular/  (click enter)

		git checkout develop-angular  (click enter)
