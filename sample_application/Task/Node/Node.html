NodeJS Configuration

We need previously installed NodeJS

Install the Test environment

Open the Windows Console 

1.-	Go to ...ls-client\src\main\javascript\test\nightwatch (click enter)
2.-	Run to add proxy configuration:
	npm config set proxy http://alpext11.proxy.corporate.ge.com:80/ (click enter)
	npm config set https-proxy http://alpext11.proxy.corporate.ge.com:80/ (click enter)
 
3.-	Run: 
	npm install (click enter)

4.-	Download selenium-standalone (selenium-server-standalone-2.39.0.jar2)from libraries:
	http://libraries.ge.com/foldersIndex.do?entity_id=29896711101&sid=101&SF=1#29896711101
	
(Note: Change the name selenium-server-standalone-2.39.0.jar2 to selenium-server-standalone-2.39.0.jar)

	
5.-	Paste selenium-standalone driver and put it in:
...ls-client\src\main\javascript\test\nightwatch\selenium-driver\ (click enter)

6.- Open Firefox (default browser) and paste the url:
	http://localhost:8080/client/javascript/app/brands/ge/#/DMC

7.-	In the console go to ...ls-client\src\main\javascript\test\nightwatch (click enter)
	and then run:
 		node nightwatch.js -c nightwatch-settings.json (click enter)
